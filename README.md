## Functions

- Filter products

  - via state
  - via city

## Clone the project

```
git@github.com:alized559/productlist.git
cd productlist
```

## Install Dependencies

```
npm i bootstrap
npm i react-bootstrap
npm i react-items-carousel
npm i axios
```

## Running

```
npm start
```
